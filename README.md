# Bank Management System

A simple Bank Management System developed in C++ that allows users to:

- Create accounts
- Deposit money
- Withdraw money
- Check account balance
- Save account details using file handling

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling

## Features

- Account Creation
- Deposit and Withdrawal
- Balance Inquiry
- Persistent Storage using Text Files
